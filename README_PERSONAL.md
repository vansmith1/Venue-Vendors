Venue Vendors (University Group Project)

This was my assignment for Year 2 Semester 1 Full Stack Development Class.

Description:
This project was developed as part of a group assignment.

Project Context:
This was a group assignment between two people, to create a website that allows the display of different venues owned by vendors, that can be viewed and booked by different hirers. 

Technologies Used
* HTML5
* CSS3
* TypeScript
* React
* Chakra UI
* React Router DOM
* React Hooks (useState, useEffect, useContext)
* Node.js

Note:
This repository contains both my individual contribution, and my partners contribution to the project. I have permission from my partner to display their code, which allows for the functionality of the website.
